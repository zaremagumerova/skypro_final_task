Git

  • git clone - клонирование существующего репозитория
  • git status - показывает состояние файлов в рабочем каталоге
  • git commit -m "Сообщение" - фиксирование изменений в репозитории
  • git push - передача изменений из локального в удаленный репозиторий
  • git pull - передача изменений из удаленного в локальный репозиторий
